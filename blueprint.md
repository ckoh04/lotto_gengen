# Lottery Gen Gen

## Overview

This web application generates random numbers for the Mega Millions and Powerball lotteries. Users can select the lottery they want to play, and the application will generate a set of numbers based on the rules of that specific lottery.

## Project Outline

### Style and Design

*   **Layout:** A clean, centered layout that is easy to use.
*   **Theme:** A dark theme inspired by the USA Mega website, with a blue gradient background and white text for high contrast.
*   **Typography:** Clear and readable fonts, with uppercase and letter-spacing for headings to create a modern look.
*   **Color Scheme:** A dark blue and black background, with white as the primary text color. The special lottery ball is styled in red, and interactive elements use a vibrant blue.
*   **Visuals:** Realistic lottery balls with a 3D effect, including a red special ball for Powerball to mimic the look of the USA Mega website.

### Features

*   **Lottery Selection:** Users can choose between "Mega Millions" and "Powerball".
*   **Number Generation:** The application will generate the appropriate number of balls with the correct number ranges for the selected lottery.
*   **Powerball Add-ons:** 
    *   **Power Play:** Users can select the Power Play option to add a multiplier to their ticket.
    *   **Double Play:** Users can select the Double Play option to enter their numbers in a second drawing.
*   **Results Display:** The generated numbers will be clearly displayed in styled lottery balls.
*   **Information Link:** A link to an external website for users to check the official results.
